# Possessive Apostrophe Rules

Apostrophes are used to indicate possession or ownership. Misusing them—particularly using them to form simple plurals—is a primary error identified by examiners.

## Singular Possession
For a single owner, the apostrophe is placed before the 's'.
- *Example*: The girl's book (one girl owns the book).

## Plural Possession
For owners ending in 's', the apostrophe is placed after the 's'.
- *Example*: The girls' book (multiple girls own the book).

## Irregular Plurals
For plural nouns that do not end in 's', the apostrophe is placed before the 's'.
- *Example*: The babies' toys.

## Special Cases
- **Names ending in 's'**: Both "James' books" and "James's books" are acceptable. However, "James's" is preferred if the second 's' is pronounced.
- **The Greengrocer's Error**: The [[greengrocers-apostrophe]] occurs when an apostrophe is used to make a word plural (e.g., "Apple's for sale"). **Apostrophes are never used to make plurals.**
