# Structural Features for Tension (AQA 2026)

The AQA 8700/1 Sample Mark Scheme (June 2026) provides a list of structural features that writers use to create tension. These features are relevant to Q3 (structural analysis) and are included in the AO2 content list.

## List of Structural Features

- **Exposition:** Introduction of setting, characters, and background information
- **Foreshadowing:** Hints or clues about future events
- **Pivotal point:** A key moment where the narrative changes direction
- **Focus shifts:** Changes in narrative focus from one character, setting, or event to another
- **First-person perspective:** Narration from a single character's point of view
- **Motifs:** Recurring images, ideas, or symbols
- **Rising action:** Events that build tension towards a climax
- **Juxtaposition:** Placing contrasting elements side by side for effect
- **Anti-climax:** A disappointing or less dramatic resolution than expected

## How to Analyse These Features

When analysing structural features for tension, consider:

- **Purpose:** Why has the writer used this feature at this point?
- **Effect:** How does this feature create or build tension?
- **Placement:** Where in the text does this feature appear?
- **Relationship:** How does this feature relate to other structural features?

## Example from the Sample Mark Scheme

The indicative standard for Q3 demonstrates analysis of foreshadowing, pivotal point, and first-person perspective in the *Life of Pi* extract. The writer uses these features to create tension around the hyena's behaviour.
