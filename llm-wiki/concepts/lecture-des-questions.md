# Lecture des Questions

Compétence présentée dans le guide BBC Bitesize pour lire et interpréter les questions d'examen. Le guide conseille de souligner les mots-clés dans les questions pour structurer les réponses et identifier les compétences requises. Cette technique est essentielle pour Paper 1, Question 2 (analyse du langage) et pour toutes les questions d'examen.
