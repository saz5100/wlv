# Fixed Thematic Prompt

A fixed thematic prompt is a single mandatory writing task with no choice between narrative and descriptive options, as used in the WLV English Equivalency test.

## Key Characteristics

- Candidates receive one prompt only, not an 'either/or' choice.
- The prompt is thematic, requiring a focused response of approximately 400 words.
- No option to switch between genres or formats.

## Implications for Revision

- Practise responding to a single prompt rather than preparing multiple genre options.
- Develop flexibility to adapt to any given theme.
- Focus on building a persuasive or descriptive arc within a single piece.
- Use structural devices such as one-word paragraphs, rhetorical questions, and clear paragraph transitions.
