# Classes de Mots

Présentation détaillée des classes de mots dans le guide BBC Bitesize pour l'analyse des choix lexicaux (Paper 1, Question 2).

## Types de classes de mots

- **Noms :** Communs, propres, abstraits.
- **Verbes :** D'action, d'état, impératifs, modaux.
- **Adjectifs :** Comparatifs, superlatifs.
- **Adverbes :** Modifient les verbes, les adjectifs ou d'autres adverbes.
- **Pronoms :** Personnels, possessifs, démonstratifs, indéfinis.
- **Modificateurs :** Mots qui modifient ou précisent le sens d'autres mots.
