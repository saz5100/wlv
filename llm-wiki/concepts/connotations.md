# Connotations

Analyse des associations d'idées au-delà du sens littéral d'un mot, présentée dans le guide BBC Bitesize avec un diagramme en araignée pour le mot « snow ». Les connotations multiples d'un mot peuvent créer des impressions contrastées.

## Exemple : « snow »

Le diagramme en araignée pour le mot « snow » inclut les associations suivantes : innocent, froid, hiver, sommeil, mort, vide, calme, sinistre, paisible, étrange, blanc.

## Exemple travaillé

L'extrait de *The Cuckoo's Calling* est utilisé pour montrer comment analyser les connotations négatives des reporters (animaux, saleté, maladie).
