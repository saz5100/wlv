# Champ Lexical

Groupe de mots liés sous un même thème, présenté dans le guide BBC Bitesize comme une technique d'analyse pour Paper 1, Question 2. L'identification d'un champ lexical permet de montrer comment un écrivain crée une atmosphère ou souligne un thème à travers le choix de vocabulaire.
