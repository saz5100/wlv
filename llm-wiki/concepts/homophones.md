# Homophones

Les homophones sont des mots qui se prononcent de la même manière mais ont des sens et souvent des orthographes différents. Homophones are words that sound the same but have different meanings and spellings. Leur confusion est une source fréquente d'erreurs dans l'écriture, and confusing these is a common error in the WLV writing task.

## Paires et groupes d'homophones courants

### General Vocabulary
- **except / accept**
    - **except** : sauf, à l'exception de. Exemple : *I did them all except the last one.*
    - **accept** : accepter. Exemple : *I accept your apology.*
- **affect / effect**
    - **affect** (verbe) : affecter, influencer. Exemple : *I don't think the weather will affect the result.*
    - **effect** (nom) : effet, résultat. Exemple : *The weather seemed to have a bad effect on everyone's mood.*
- **aloud / allowed**
    - **aloud** : à voix haute. Exemple : *Mo really likes reading aloud in class.*
    - **allowed** : autorisé. Exemple : *Nobody is allowed in here at lunchtime.*
- **write / right**
    - **write** : écrire. Exemple : *I'll write a letter and explain.*
    - **right** : correct, droit. Exemple : *Nobody got the right answer.*
- **complement / compliment**
    - **complement** : complète quelque chose (completes something).
    - **compliment** : un compliment, un éloge (a piece of praise).
- **stationary / stationery**
    - **stationary** : immobile (not moving).
    - **stationery** : papeterie, matériel d'écriture (writing materials).

### Pronouns and Contractions
- **who's / whose**
    - **who's** : contraction de 'who is' ou 'who has'. Exemple : *Tell me who's going and then I'll decide.*
    - **whose** : possessif de 'who'. Exemple : *He couldn't return it because he didn't know whose coat it was.*
- **your / you're**
    - **your** : possessif de 'you'. Exemple : *I asked your sister to bring it.*
    - **you're** : contraction de 'you are'. Exemple : *You're not going out like that.*
- **there / they're / their**
    - **there** : là, il y a (location). Exemple : *There are twenty-five people in the class.*
    - **they're** : contraction de 'they are'. Exemple : *They're not sitting in the right places.*
    - **their** : possessif de 'they'. Exemple : *They have all done their homework.*
- **where / wear / we're**
    - **where** : où. Exemple : *We have no idea where it is.*
    - **wear** : porter (vêtement), user. Exemple : *Turn it off or it will wear out.*
    - **we're** : contraction de 'we are'. Exemple : *We're going anyway.*

### Common Confusions
- **past / passed**
    - **past** : passé, au-delà de. Peut être un nom, un adjectif ou une préposition. Exemple : *He walked right past me as if I wasn't there.*
    - **passed** : passé (verbe 'to pass'). C'est toujours un verbe (past tense of pass). Exemple : *I passed him in the street an hour ago.*
- **to / too / two**
    - **to** : préposition (vers, pour). Exemple : *There were only two to sit.*
    - **too** : aussi, trop. Exemple : *That was one too many.*
    - **two** : le nombre 2. Exemple : *There were only two to sit.*
- **practice / practise**
    - **practice** (nom) : pratique, entraînement. Exemple : *If you don't go to the practice exams you'll be left out of the team.*
    - **practise** (verbe) : pratiquer, s'entraîner. Exemple : *If you want to improve you will have to practise every day.*
