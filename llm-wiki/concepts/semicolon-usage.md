# Semicolon Usage

A semicolon (;) is a punctuation mark that is "stronger" than a comma but "weaker" than a full stop.

## Primary Uses
1. **Connecting Independent Clauses**: Used to link two closely related independent clauses without using a conjunction.
   - *Example: "The rain was relentless; the streets were flooded."*
2. **Complex Lists**: Used to separate items in a list when the items themselves already contain commas.
   - *Example: "I have lived in Paris, France; Berlin, Germany; and Rome, Italy."*

## Technical Requirement
A semicolon must always be preceded by an **independent clause**.
