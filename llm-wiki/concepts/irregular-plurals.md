# Pluriels irréguliers

Les pluriels irréguliers sont des noms qui ne suivent pas la règle standard d'ajout de '-s' ou '-es' pour former leur pluriel. Beaucoup de ces pluriels sont irréguliers car ils sont dérivés du latin ou du grec. Leur maîtrise est essentielle pour une orthographe correcte.

## Règles générales pour les pluriels

1. **Ajout de '-s'** : la plupart des noms (pizza $\rightarrow$ pizzas, donkey $\rightarrow$ donkeys, birthday $\rightarrow$ birthdays).
2. **Ajout de '-es'** : noms se terminant par -s, -sh, -ch, -x, -z (latch $\rightarrow$ latches, church $\rightarrow$ churches).
3. **Noms se terminant par -o** : certains prennent '-es' (tomato $\rightarrow$ tomatoes, mosquito $\rightarrow$ mosquitoes).
4. **Noms se terminant par -y** : 
    - Consonne + y $\rightarrow$ '-ies' (quality $\rightarrow$ qualities, family $\rightarrow$ families).
    - Voyelle + y $\rightarrow$ '-s' (donkey $\rightarrow$ donkeys).
5. **Noms invariables** : sheep $\rightarrow$ sheep.
6. **Noms irréguliers (changement de forme)** : woman $\rightarrow$ women.
7. **Noms d'origine grecque ou latine** : utilisent souvent des terminaisons spécifiques (ex: hypothesis $\rightarrow$ hypotheses).

## Liste des pluriels irréguliers courants

| Singulier | Pluriel | Règle / Origine |
|-----------|---------|-----------------|
| pizza | pizzas | Ajout de '-s' (régulier) |

| donkey | donkeys | Ajout de '-s' (régulier, voyelle + y) |
| stadium | stadiums | Ajout de '-s' (forme anglicisée) |
| latch | latches | Ajout de '-es' après 'ch' |
| church | churches | Ajout de '-es' après 'ch' |
| tomato | tomatoes | Ajout de '-es' après 'o' |
| mosquito | mosquitoes | Ajout de '-es' après 'o' |
| quality | qualities | 'y' $\rightarrow$ 'ies' (consonne + y) |
| family | families | 'y' $\rightarrow$ 'ies' (consonne + y) |
| soliloquy | soliloquies | 'y' $\rightarrow$ 'ies' (consonne + y) |

| sheep | sheep | Invariable |
| hypothesis | hypotheses | Grec : '-is' $\rightarrow$ '-es' |
| parenthesis | parentheses | Grec : '-is' $\rightarrow$ '-es' |

## Pluriels d'origine latine et grecque

Certains mots conservent des formes plurielles issues directement de leurs racines classiques :

- **Analysis** $\rightarrow$ **Analyses**
- **Thesis** $\rightarrow$ **Theses**
- **Criterion** $\rightarrow$ **Criteria**
- **Datum** $\rightarrow$ **Data**
- **Matrix** $\rightarrow$ **Matrices**
- **Phenomenon** $\rightarrow$ **Phenomena**
