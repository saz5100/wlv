# Répétition comme Pattern

Technique d'analyse présentée dans le guide BBC Bitesize qui consiste à identifier la répétition de mots, de types de mots ou de structures comme un pattern significatif. Cette analyse permet de montrer comment un écrivain crée de l'emphase, établit un rythme ou souligne un thème à travers le texte.
