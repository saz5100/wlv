# Séquencement Chronologique

Présentation du séquencement chronologique dans le guide BBC Bitesize pour Paper 1, Question 3 (analyse de la structure). Le séquencement chronologique présente les événements dans l'ordre temporel, ce qui permet de développer les personnages et la tension de manière progressive. Le guide note que le séquencement chronologique peut également refléter l'ennui ou la monotonie d'un personnage.
