# Grade 9 Analytical Phrase Bank

Using sophisticated phrases signals to the examiner that you are working at a high level. This bank organises phrases by exam skill.

## Starting a Paragraph / Introducing Ideas

| Phrase | Best for |
|--------|----------|
| Firstly, the writer... | Any opening |
| First of all... | Q2, Q3 opening |
| To start with... | Q2 opening |
| One technique worth mentioning is... | Q2 language analysis |
| The writer immediately establishes... | Q3 structure analysis |
| From the outset, the writer... | Q3 structure analysis |
| It is important to point out that... | Adding emphasis |

## Adding Information

| Phrase | Best for |
|--------|----------|
| Moreover... | Building a point |
| Furthermore... | Building a point |
| In addition... | Adding a second example |
| Additionally... | Adding a second example |
| What's more... | Informal but effective |
| This is reinforced by... | Connecting two techniques |
| This idea is further developed when... | Showing progression |

## Language Analysis Phrases

| Phrase | Effect demonstrated |
|--------|-------------------|
| The writer employs [technique] to... | Identifies method |
| The use of [word/phrase] conveys... | Shows understanding of effect |
| This evokes a sense of... | Reader response |
| The connotations of [word] suggest... | Word-level analysis |
| This forces the reader to confront... | Powerful reader effect |
| The writer deliberately chooses... | Writer's intentionality |
| Through this [technique], the writer positions the reader to feel... | Sophisticated analysis |
| The semantic field of [topic] is established through... | Thematic awareness |

## Structure Analysis Phrases

| Phrase | Effect demonstrated |
|--------|-------------------|
| At the beginning of the text, the writer focuses on... | Opening analysis |
| The writer then shifts the reader's attention to... | Focus shift |
| This contrast between [X] and [Y] highlights... | Comparative structure |
| The pace quickens as... | Pace analysis |
| The paragraph length contracts to... | Structural technique |
| The cyclical structure — returning to [image/idea] — emphasises... | Cyclical structure |
| The writer withholds [information] until... | Delayed revelation |
| The focal point narrows from... to... | Zoom technique |
| The narrative perspective shifts from... to... | Perspective shift |

## Concluding Phrases

| Phrase | Best for |
|--------|----------|
| Overall, the writer's use of... | Any summary |
| In conclusion, the combination of... | Q2, Q3 summary |
| Ultimately, the effect created is... | Emphasising cumulative effect |
| The writer successfully [achieves effect] by... | Concluding judgement |

## Creative Writing Phrases (Q5)

| Phrase | Purpose |
|--------|---------|
| The air was thick with... | Sensory description |
| A palpable sense of [emotion] hung over... | Atmosphere |
| The silence was broken only by... | Sound focus |
| [Object] loomed like a... | Simile + personification |
| Time seemed to slow as... | Pace manipulation |
| Every detail — the [x], the [y], the [z] — suggested... | Cumulative detail |
| Nothing moved except... | Stillness with contrast |
| Then, without warning,... | Sudden shift |

## Sophisticated Vocabulary for Analysis

| Instead of... | Use... |
|-------------|--------|
| Shows | Conveys, suggests, evokes, implies, demonstrates |
| Makes the reader feel | Positions the reader to, forces the reader to, invites the reader to |
| The writer uses | The writer employs, deploys, utilises, manipulates |
| This means | This signifies, this indicates, this suggests |
| A feeling of | A sense of, an atmosphere of, an impression of |
| Good/bad | Powerful, striking, unsettling, evocative, haunting, arresting |
