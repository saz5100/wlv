# Reflexive Pronouns

Reflexive pronouns (e.g., *myself*, *himself*, *themselves*) are used when the object of a sentence is the same as the subject.

## Correct Usage
Reflexive pronouns should only be used when preceded by a reflexive verb or specific prepositions.
- *Correct*: "I hurt **myself** while cooking."
- *Correct*: "He decided to treat **himself** to a coffee."

## The "Pompous" Error
A common error is using 'myself' as a substitute for 'I' or 'me' in an attempt to sound more formal. This is grammatically incorrect and can appear "pompous and egotistical".
- *Incorrect*: "Please send the report to **myself**."
- *Correct*: "Please send the report to **me**."
- *Incorrect*: "**Myself** and Sarah will attend the meeting."
- *Correct*: "Sarah and **I** will attend the meeting."
