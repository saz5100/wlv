# Inférence

Compétence de lecture présentée dans le guide BBC Bitesize qui consiste à lire entre les lignes pour déduire des informations non explicitement énoncées. L'inférence est une compétence clé pour de nombreuses questions d'examen, pas seulement pour Paper 1, Question 2.

## Phrases clés pour l'inférence

- « this suggests »
- « this implies »
- Langage hypothétique : « might », « maybe », « perhaps »

## Exemple travaillé

Le guide utilise un extrait non identifié (Source E, famille) pour montrer comment déduire des informations sur les personnages et leurs relations à partir d'indices textuels.
