# Condensed Assessment Layout

The condensed assessment layout refers to the WLV English Equivalency test's 2-hour format, which strips out several components of the full AQA GCSE English Language specification.

## Components Removed

- Paper 2 comparison and synthesis
- Q4 evaluation essay
- Spoken Language Endorsement
- Writing prompt choice

## Components Retained

- Section A: Reading (25 marks) — Q1, Q2, Q3 skills on a single extract
- Section B: Writing (25 marks) — Q5-style extended writing with a single prompt

## Implications for Revision

- Revision time should be concentrated on the retained components only.
- Full mock papers are unnecessary; targeted practice of Q1, Q2, Q3, and Q5 is sufficient.
- The condensed format allows for deeper focus on SPaG, which carries 50% of the writing marks.
