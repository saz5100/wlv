# AO1 Multiple-Choice Format (AQA 2026)

The AQA 8700/1 Sample Mark Scheme (June 2026) introduces a significant change to Question 1 (AO1). Q1 is now a 4-mark multiple-choice section rather than the traditional short-answer written response format.

## Key Features

- **Format:** Four separate multiple-choice questions
- **Response method:** Shade the circle (or equivalent)
- **Marks:** 4 marks (one mark per question)
- **Assessment Objective:** AO1 (identify and interpret explicit and implicit information and ideas)

## Comparison with Previous Format

| Feature | Previous Format | New Format (2026) |
|---------|-----------------|-------------------|
| Response type | Written short answers | Multiple-choice |
| Number of items | 4 | 4 |
| Marks | 4 | 4 |
| Marking | Examiner judgement | Objective (correct/incorrect) |
| Time required | More | Less |

## Implications for Students

- **No writing required:** Students do not need to write full sentences for Q1
- **Speed:** Q1 can be completed more quickly, allowing more time for other questions
- **Accuracy:** Students must read the questions carefully to select the correct option
- **No partial credit:** Each question is either correct or incorrect

## Implications for the WLV Equivalency Test

The WLV English Equivalency test may or may not adopt this format. Students should confirm the format of their specific test. If the WLV test retains the traditional written format, students should practise both formats.
