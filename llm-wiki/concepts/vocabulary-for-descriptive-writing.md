# Vocabulary for Descriptive Writing

A well-chosen word can transform a simple description into a powerful one. Use these word banks to add precision and impact to your Q5 writing.

## Weather & Light

| Atmosphere | Words to use |
|-----------|-------------|
| Bright sun | blazing, scorching, dazzling, relentless, blinding, harsh, molten, glaring |
| Cloudy | overcast, leaden, grey, brooding, oppressive, heavy, low-hanging |
| Rain | torrential, relentless, drizzling, lashing, streaming, pelting, sheet of |
| Storm | raging, furious, howling, violent, tempestuous, ferocious, merciless |
| Fog/mist | thick, swirling, impenetrable, ghostly, pale, drifting, clinging, hazy |
| Cold | biting, bitter, numbing, piercing, sharp, frosty, icy, raw |
| Darkness | impenetrable, absolute, suffocating, velvety, deep, vast, swallowing |

## Settings & Places

| Setting | Words to use |
|---------|-------------|
| City | bustling, chaotic, vibrant, decaying, sprawling, cramped, towering, labyrinthine |
| Rural | peaceful, sprawling, rolling, barren, lush, untouched, desolate, windswept |
| Abandoned | derelict, decaying, crumbling, rusting, overgrown, hollow, skeletal, silent |
| Indoors | cluttered, sparse, cosy, sterile, grand, decaying, cramped, airless |
| Water | churning, still, murky, crystalline, stagnant, frothing, glassy, rippling |
| Woodland | dense, tangled, shadowy, dappled, whispering, ancient, mossy, looming |

## Sensory Words — Sound

| Type | Words |
|------|-------|
| Loud | thunderous, deafening, piercing, shrill, blaring, roaring, shattering |
| Soft | whispered, murmured, faint, distant, muted, hushed, gentle, soft |
| Sudden | crashing, slamming, cracking, snapping, banging, shattering |
| Continuous | droning, humming, buzzing, rumbling, howling, grinding, hissing |
| Human | gasping, sobbing, chuckling, sighing, groaning, muttering, shrieking |

## Sentence Starters for Description

| Purpose | Starter |
|---------|---------|
| Wide shot | "The first thing that struck me was..." |
| Zoom in | "Closer, you could see..." |
| Sound | "The only sound was..." |
| Smell | "The air smelled of..." |
| Touch | "Underfoot, the [surface] felt..." |
| Sudden change | "Then, without warning..." |
| Contrast | "But beneath this [calm/beauty] lay..." |
| Final reflection | "And then, as if on cue..." |

## Using This Vocabulary Effectively

* **Don't overuse** — one powerful word every 2-3 sentences is enough
* **Context is everything** — "blazing sun" works in a desert; "frosty" works in winter
* **Combine the senses** — describe what you see, hear, smell, and feel
* **Precise is better than fancy** — "crumbling" is better than "broken and falling apart"
