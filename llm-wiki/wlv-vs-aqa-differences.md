# WLV vs AQA Differences

The WLV English Equivalency test is a condensed version of AQA GCSE English Language Paper 1 (8700). This page outlines what is kept and what is removed.

## What Is Kept (from AQA Paper 1)

| Component | AQA Equivalent | Marks |
|-----------|----------------|-------|
| Explicit retrieval | Q1 | 4 |
| Language analysis | Q2 | 8 |
| Structural analysis | Q3 | 8 |
| Creative/extended writing | Q5 | 25 (including SPaG) |

## What Is Removed

| Component | Reason for Removal |
|-----------|-------------------|
| Paper 2 comparison and synthesis | WLV uses a single source extract only |
| Evaluation essay (Q4) | Stripped from the 2-hour format |
| Spoken Language Endorsement | Not part of the on-campus assessment |
| Writing prompt choice | Single fixed thematic prompt only |

## Key Structural Differences

- **Single source extract** — Unlike the full GCSE, which uses two texts for Paper 2, the WLV test uses only one unseen prose extract.
- **No Q4 evaluation** — The 20-mark evaluation essay is entirely absent.
- **Fixed writing prompt** — Candidates receive one mandatory prompt, not a choice between narrative and descriptive options.
- **50% SPaG weighting** — Technical accuracy accounts for half of the writing marks, making it a higher priority than in the full GCSE context.

## Implications for Revision

- Focus revision exclusively on Q1, Q2, Q3, and Q5 skills from AQA Paper 1 past papers.
- Do not practise Q4 evaluation essays or Paper 2 synthesis tasks.
- Prioritise SPaG practice, particularly comma splicing, apostrophes, and sentence demarcation.
- Prepare for a single writing prompt rather than practising multiple genre options.
