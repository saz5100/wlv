# All The Light We Cannot See

Roman de la Seconde Guerre mondiale utilisé comme exemple travaillé dans le guide BBC Bitesize pour l'analyse structurelle avancée (Paper 1, Question 3). L'extrait, décrivant une mission de bombardement américaine, montre comment un paragraphe court d'un seul mot (« France ») crée un effet dramatique et perturbant.
