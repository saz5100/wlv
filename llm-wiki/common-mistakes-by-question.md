# Common Mistakes by Question

## Q1 -- Retrieval (4 marks)

| Mistake | Consequence | Fix |
|---------|------------|-----|
| Writing full sentences | Wastes time | Just list 4 clear facts |
| Inferring instead of retrieving | 0 marks for made-up info | Only write what is explicitly stated |
| Copying huge chunks | No time saved | Short quotes or paraphrases |
| Writing 3 instead of 4 points | Lose 1 mark | Always count your points |

## Q2 -- Language Analysis (8 marks)

| Mistake | Consequence | Fix |
|---------|------------|-----|
| Feature spotting without analysis | Stuck at Level 2 | Always explain the EFFECT |
| Not quoting from the specified lines | Marks deducted | Only use the line range given |
| Describing plot instead of language | Entirely off-task | Focus on HOW the writer writes |
| Vague terminology ("the word creates an effect") | No precision | Use exact terms: simile, metaphor, semantic field |
| Only one or two paragraphs | Not enough depth | Aim for 3 well-developed paragraphs |

## Q3 -- Structure Analysis (8 marks)

| Mistake | Consequence | Fix |
|---------|------------|-----|
| Analysing language not structure | Entirely off-task | Only discuss: order, pace, focus, beginnings/endings |
| Only discussing one paragraph | Cannot achieve top band | Refer to beginning, middle, AND end |
| Describing the plot | No analysis | Analyse WHY the writer chose that order |
| Not quoting structural evidence | No proof | Quote beginnings/endings, shifts, paragraph changes |

## Q4 -- Evaluation (20 marks) OUT OF SCOPE for WLV

For AQA reference only: analyse, evaluate, agree/disagree with the given statement using evidence.

## Q5 -- Creative Writing (40 marks)

| Mistake | Consequence | Fix |
|---------|------------|-----|
| No planning | Wandering plot, unfinished | Spend 5 mins on a plan |
| Writing under 2 pages | Cannot achieve top marks | Aim for 2.5-3 pages |
| No paragraphs | Capped at Level 2 for AO5 | Use TIP-TOP: Time, Place, Topic, Person change = new paragraph |
| Telling not showing | Flat, unengaging writing | Use sensory details and actions |
| Repetitive vocabulary | Low AO5 marks | Use a range of precise vocabulary |
| SPaG errors | Lose up to 16 marks | Save 5 mins to proofread |
| Wrong tone/register | Inappropriate for task | Match the prompt's mood |
| No ending/rushed ending | Unsatisfying conclusion | Plan your ending before you start |

## General Mistakes

| Mistake | Fix |
|---------|-----|
| Misreading the question | Underline keywords, re-read twice |
| Not managing time | Stick to timings -- use a watch |
| Panicking over unfamiliar text | It's designed to be unseen -- apply your skills |
| Not re-reading answers | Catch silly mistakes in the last 5 mins |
