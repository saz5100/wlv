## Test Format

### Paper 1: Reading and Comprehension (1 hour 30 minutes)

#### Section A: Reading for Meaning
| Skill | Description |
|-------|-------------|
| Explicit information | Locating facts and details from texts |
| Implicit information | Inference and deduction â€” reading between the lines |
| Purpose and audience | Identifying who the text is for and why it was written |
| Summarising and paraphrasing | Condensing information in your own words |

#### Section B: Language Analysis
| Skill | Description |
|-------|-------------|
| Word classes | Nouns, verbs, adjectives, adverbs, prepositions |
| Language choices | Vocabulary, imagery, figurative language (simile, metaphor, personification) |
| Creating meaning | How language choices affect the reader |

#### Section C: Comparison
| Skill | Description |
|-------|-------------|
| Comparing texts | Similarities and differences between two passages |
| Comparing language | How each text uses language differently |
| Comparing tone and purpose | Different approaches by different writers |

### Paper 2: Writing (1 hour 30 minutes)

#### Section A: Grammar and Mechanics
| Skill | Description |
|-------|-------------|
| Sentence structure | Simple, compound, and complex sentences |
| Punctuation | Full stops, commas, apostrophes, speech marks, colons, semicolons |
| Spelling | Common patterns, homophones, prefixes, suffixes |
| Paragraphing | Topic sentences, cohesive devices |

#### Section B: Written Communication
| Skill | Description |
|-------|-------------|
| Writing for different purposes | Inform, explain, persuade, argue |
| Writing for different audiences | Formal vs informal register |
| Organisation and structure | Introductions, conclusions, logical flow |
| Register and tone | Appropriate language for purpose and audience |

## Scoring

## Preparation Strategies
1. **Read widely:** Newspapers (The Guardian, The Times), articles, opinion pieces
2. **Practice reading comprehension:** Summarise paragraphs after reading
3. **Language analysis:** Annotate texts identifying word classes and figurative language
4. **Writing practice:** Write on different topics for different purposes
5. **Timed practice:** Complete full papers under timed conditions
6. **Grammar revision:** Review punctuation and sentence structure rules
7. **Time:** 20â€“40 hours preparation typically sufficient
