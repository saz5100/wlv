# Handwritten Response Strategies

The University of Worcester English Equivalency test requires Section B (Writing) to be handwritten and then scanned or photographed for submission. This differs from typed submissions and requires specific preparation.

## Why Handwriting Matters

- **Legibility**: Examiners must be able to read your response
- **Pacing**: Handwriting is slower than typing — time management is crucial
- **Editing**: Corrections are more difficult to make neatly
- **Presentation**: Clear presentation is part of the assessment criteria

## Preparation Strategies

### 1. Practise Handwriting Under Timed Conditions
- Set a timer for 1 hour (the recommended time for Section B)
- Write a full response by hand
- Evaluate legibility and speed

### 2. Develop a Planning System
- Spend 5–10 minutes planning before writing
- Use a separate sheet for notes and planning
- Create a brief outline with key points

### 3. Write Neatly
- Use a pen you are comfortable with
- Write in a consistent size and style
- Leave space between paragraphs
- Avoid crossing out heavily — use a single line through errors

### 4. Manage Your Time
- Allocate time for planning, writing, and checking
- Aim to finish with 5–10 minutes for proofreading
- If running short, prioritise completing the response over perfection

## Scanning and Submission

### Equipment
- Smartphone camera or scanner
- Good lighting
- Flat surface

### Tips for Clear Scans
- Place the page on a flat, well-lit surface
- Hold the camera directly above the page
- Ensure the entire page is in frame
- Check the image is in focus before submitting
- Use a scanning app (e.g., Adobe Scan, CamScanner) for best results

### File Format
- Check the test instructions for accepted formats (usually PDF or JPEG)
- Name the file clearly (e.g., 'YourName_SectionB.pdf')
- Email to the provided address promptly

## Common Pitfalls

- **Illegible handwriting**: Practise to ensure readability
- **Running out of time**: Practise timed writing regularly
- **Poor planning**: Always plan before writing
- **Messy corrections**: Use a single line through errors
- **Blurry scans**: Check image quality before submitting
- **Late submission**: Allow time for scanning and emailing
