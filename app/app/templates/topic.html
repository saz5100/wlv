📖 Key Terminology for This Topic

Exam-grade terms for Memory and Storage.

Volatile
Memory that loses its contents when power is turned off. RAM is volatile.
Non-Volatile
Memory that retains data when power is off. ROM, HDD, and SSD are non-volatile.
Defragmentation
Reorganising data on an HDD so related data is stored contiguously, improving read speed.
Flash Memory
A type of non-volatile memory used in SSDs. No moving parts, fast access times.
Firmware
Software permanently stored in ROM, such as BIOS/UEFI. Provides low-level hardware control.
Virtual Memory
Using part of the HDD/SSD as extra RAM when physical RAM is full. Slower than real RAM.

📖 Key Terminology for This Topic

Exam-grade terms for Network Security.

Malware
Malicious software designed to harm a computer system. Includes viruses, worms, and trojans.
Phishing
A social engineering attack where fake emails trick users into revealing personal information.
Firewall
Software or hardware that monitors and controls incoming and outgoing network traffic based on security rules.
Encryption
Scrambling data so it can only be read by someone with the correct decryption key.
SQL Injection
An attack where malicious SQL code is inserted into input fields to manipulate a database.
Penetration Testing
Authorised simulated attacks on a system to identify security vulnerabilities before criminals do.
