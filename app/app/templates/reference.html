🏗️ Von Neumann Architecture ▼
Von Neumann Architecture — CPU, Memory, Buses, and I/O
Key Points:
  • Von Neumann architecture stores both data and instructions in the same memory
  • The Fetch-Decode-Execute cycle is the fundamental process by which the CPU processes instructions
  • Three buses connect CPU to memory: Data Bus (carries data), Address Bus (carries memory addresses), Control Bus (carries control signals)
  • Registers are fast, temporary storage locations inside the CPU

📖 Key Terminology

Exam-grade terms from the reference tables above.

Volatile
Memory that loses its contents when power is turned off. RAM is volatile; ROM is non-volatile.
Non-Volatile
Memory that retains data when power is off. ROM, HDD, SSD are non-volatile.
Defragmentation
Reorganising data on an HDD so related data is stored contiguously, improving read speed. Not needed on SSDs.
Time Complexity
A measure of how the runtime of an algorithm grows as input size increases. Expressed as Big O notation (e.g. O(n), O(n²), O(log n)).
Stable Sort
A sorting algorithm that preserves the relative order of equal elements. Bubble, merge, and insertion sorts are stable.
Topology
The physical or logical arrangement of devices in a network. Common topologies: star, mesh, bus.
Backbone
The main cable in a bus topology that connects all devices. If the backbone fails, the entire network goes down.
Firmware
Software permanently stored in ROM, such as BIOS/UEFI. It provides low-level control for hardware.
In-Place Algorithm
An algorithm that sorts data using only a constant amount of extra memory. Bubble and insertion sorts are in-place; merge sort is not.
Recursion
A technique where a function calls itself to solve smaller instances of the same problem. Merge sort uses recursion.

1. Which type of memory is volatile and stores running programs and data?
✅ Correct! RAM is volatile and holds the programs and data currently in use. It loses its contents when power is turned off.
2. Which storage device has no moving parts and is the fastest for everyday use?
✅ Correct! SSDs use flash memory with no moving parts, making them faster, more durable, and more power-efficient than HDDs.
3. Which sorting algorithm has a time complexity of O(n log n) in all cases?
✅ Correct! Merge Sort has O(n log n) time complexity in the best, average, and worst cases. It is a divide-and-conquer algorithm.
4. In a star topology, what happens if the central switch fails?
✅ Correct! In a star topology, all devices connect through a central switch. If the switch fails, no device can communicate.
5. Which search algorithm requires the data to be sorted first?
✅ Correct! Binary search repeatedly halves the search space, which only works correctly on sorted data.
6. What does defragmentation do on an HDD?
✅ Correct! Defragmentation rearranges fragmented data so related pieces are stored close together, reducing read times on HDDs.
7. Which network topology has the highest reliability because data can take multiple paths?
✅ Correct! In a mesh topology, every device connects to every other device, providing multiple redundant paths.
8. What is the main difference between RAM and ROM?
✅ Correct! RAM is volatile (loses data on power off) and holds running programs. ROM is non-volatile and stores firmware like boot instructions.
